## Test Scenarios
1. **Happy Path Complete Flow**: New user → auth → onboarding → service creation → dashboard view
2. **Existing User Flow**: Returning user → auto-login → data loading → normal usage
3. **Network Interruption**: User performing action → network drops → retry → success
4. **Authentication Expiry**: Long session → token expires → re-auth → continue
5. **Data Synchronization**: Multiple operations → all data consistent across screens
6. **Error Recovery**: Various API errors → user can recover and continue
7. **Performance Under Load**: Heavy usage → app remains responsive
