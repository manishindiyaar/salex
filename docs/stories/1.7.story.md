# Story 1.7: Display Daily Analytics

## Status
Ready for Review

## Story
**As a** salon owner, **I want** to see my total bookings and total revenue for the day at a glance, **so that** I can quickly understand my daily business performance.

## Acceptance Criteria
1. A new "Analytics" or "Report" tab is available in the merchant app's main navigation.
2. The screen makes an authenticated call to a new API endpoint (e.g., `GET /api/v1/businesses/{id}/analytics/daily`).
3. The API calculates the total number of bookings with a status of `CONFIRMED` or `COMPLETED` for the current calendar day.
4. The API calculates the sum of the `price` for all `CONFIRMED` or `COMPLETED` bookings for the current calendar day.
5. The screen clearly displays two metrics: "Today's Bookings" and "Today's Revenue".
6. The calculations must be based on the salon's local timezone.

```typescript
  };
```

**Database Query Pattern:**
```sql
SELECT 
  COUNT(*) as totalBookings,
  SUM(s.price) as totalRevenue
FROM "Booking" b
JOIN "Service" s ON b."serviceId" = s.id
WHERE b."businessId" = $1 
  AND b.status IN ('CONFIRMED', 'COMPLETED')
  AND DATE(b."bookingTime" AT TIME ZONE $2) = $3
```

### Backend Architecture Context [Source: architecture/12-backend-architecture.md]

**Analytics Module Structure:**
```typescript
@Injectable()
export class AnalyticsService {
  constructor(private prisma: PrismaService) {}

  async getDailyAnalytics(
    businessId: string,
    ownerId: string,
    date?: string,
    timezone?: string
  ): Promise<DailyAnalytics> {
    // Validate business ownership
    await this.validateBusinessOwnership(businessId, ownerId);
    
    // Calculate date range for current calendar day in timezone
    const targetDate = date ? new Date(date) : new Date();
    const businessTimezone = timezone || 'UTC'; // Default or get from business profile
    
    // Query bookings with revenue calculation
    const result = await this.prisma.booking.aggregate({
      where: {
        businessId,
        status: { in: ['CONFIRMED', 'COMPLETED'] },
        bookingTime: {
          gte: startOfDay(targetDate, businessTimezone),
          lt: endOfDay(targetDate, businessTimezone),
        },
      },
      _count: true,
      _sum: {
        service: {
          price: true,
        },
      },
    });
    
    return {
      businessId,
      date: format(targetDate, 'yyyy-MM-dd'),
      totalBookings: result._count,
      totalRevenue: result._sum.service?.price || 0,
      timezone: businessTimezone,
    };
  }

  private async validateBusinessOwnership(businessId: string, ownerId: string): Promise<void> {
    const business = await this.prisma.business.findFirst({
      where: { id: businessId, ownerId },
    });
    
    if (!business) {
      throw new ForbiddenException('Access denied: Business not found or not owned by user');
    }
  }
}
```

```typescript
// apps/merchant-app/src/screens/AnalyticsScreen.tsx
export const AnalyticsScreen = () => {
  const [analytics, setAnalytics] = useState<DailyAnalytics | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);
  
  const { businessId } = useAuthStore(); // Get from existing Zustand store
  
  useEffect(() => {
    fetchDailyAnalytics();
  }, []);
  
  const fetchDailyAnalytics = async () => {
    try {
      setLoading(true);
      const response = await apiClient.get(`/businesses/${businessId}/analytics/daily`);
      setAnalytics(response.data.data);
    } catch (err) {
      setError('Failed to load analytics');
    } finally {
      setLoading(false);
    }
  };
  
  return (
    <ScrollView style={styles.container}>
      <Card style={styles.card}>
        <Card.Title title="Today's Performance" />
        <Card.Content>
          <Text variant="headlineMedium">{analytics?.totalBookings || 0}</Text>
          <Text variant="bodyMedium">Today's Bookings</Text>
          
          <Text variant="headlineMedium">${analytics?.totalRevenue?.toFixed(2) || '0.00'}</Text>
          <Text variant="bodyMedium">Today's Revenue</Text>
        </Card.Content>
      </Card>
    </ScrollView>
  );
};
```

### Analytics Calculation Logic

**Daily Revenue Calculation:**
1. Filter bookings by businessId and current calendar day
2. Include only CONFIRMED and COMPLETED booking statuses
3. Join with Service table to get pricing information
4. Sum all service prices for matching bookings
5. Handle timezone conversion for accurate day boundaries

**Daily Bookings Count:**
1. Count all bookings matching businessId and date criteria
2. Filter by CONFIRMED and COMPLETED statuses only
3. Ensure timezone-aware date range calculation

**Timezone Handling:**
- Use business's local timezone for day boundary calculations
- Convert UTC database timestamps to business timezone
- Ensure consistent date range queries across different timezones
