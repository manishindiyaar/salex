# Story 1.5: Manage Business Hours (Backend Focus)

## Status
Done

## Story
**As a** salon owner, **I want** to set and update my weekly business hours through API endpoints, **so that** the system knows when my salon is open for bookings and can calculate available time slots accordingly.

## Acceptance Criteria
1. **Business Hours API**: PUT endpoint allows business owners to update their complete weekly schedule in the hoursOfOperation JSONB field
2. **Hours Retrieval API**: GET endpoint returns current business hours in a structured format for the authenticated business
3. **Time Slot Calculation**: Service method calculates available booking slots based on business hours and existing bookings
4. **Data Validation**: Hours format validation ensures proper time format (HH:MM) and logical open/close times
5. **Authorization Security**: Only business owners can access or modify their own business hours
6. **API Testing**: Comprehensive test suite validates hours management and time slot calculation

### Backend Architecture Context [Source: architecture/12-backend-architecture.md]

**Hours Management Service Pattern:**
```typescript
@Injectable()
export class BusinessService {
  async updateBusinessHours(
    businessId: string,
    ownerId: string,
    hoursDto: UpdateBusinessHoursDto
  ): Promise<Business> {
    // Validate business ownership
    await this.validateBusinessOwnership(businessId, ownerId);
    
    // Validate hours format and logic
    this.validateBusinessHours(hoursDto.hoursOfOperation);
    
    // Update business record
    return this.prisma.business.update({
      where: { id: businessId },
      data: { hoursOfOperation: hoursDto.hoursOfOperation },
      include: { salon: true }
    });
  }
}

@Injectable()
export class TimeSlotsService {
  async getAvailableSlots(
    businessId: string,
    serviceId: string,
    dateRange: { start: string; end: string }
  ): Promise<TimeSlot[]> {
    // Get business hours
    const business = await this.getBusinessWithHours(businessId);
    
    // Get service duration
    const service = await this.getService(serviceId);
    
    // Get existing bookings
    const bookings = await this.getBookingsInRange(businessId, dateRange);
    
    // Calculate available slots
    return this.calculateAvailableSlots(business.hoursOfOperation, service, bookings, dateRange);
  }
}
```

### Time Slot Calculation Logic

**Available Slots Algorithm:**
1. Parse business hours for requested date range
2. Generate time slots based on business hours and service duration
3. Filter out slots that conflict with existing bookings
4. Return available slots with metadata (date, time, duration)

**Slot Interval Configuration:**
- Default: 30-minute intervals
- Configurable based on service type
- Minimum: 15 minutes, Maximum: 120 minutes
