# Story 2.1: Generate & Display Business QR Code

**Epic**: The Core Booking Loop  
**Story ID**: 2.1  
**Priority**: High  
**Estimated Effort**: 8 story points  
**Status**: Draft

## Story

**As a** salon owner, **I want** to view and share a unique QR code for my business from the app, **so that** my customers can easily scan it to start a booking.

## Acceptance Criteria

### AC1: QR Code Generation Service
- The backend generates a unique QR code upon business creation or request
- QR code contains WhatsApp deep link with format: `https://wa.me/{whatsappNumber}?text=BOOK_AT_{businessId}`
- QR code is stored as base64 image data or file reference in the database
- QR code generation service handles business routing code integration

### AC2: QR Code Display Screen
- A new screen or section in the merchant app displays the business QR code
- QR code is rendered clearly and at appropriate size for scanning
- Screen includes business name and explanation text for customers
- Loading state while QR code is being generated or retrieved

### AC3: QR Code Sharing Functionality
- The merchant has an option to share the QR code image from the app
- Share functionality uses platform-native sharing (iOS/Android)
- QR code can be saved to device photo gallery
- Share includes descriptive text explaining how customers should use it

### AC4: WhatsApp Deep Link Integration
- When scanned, QR code opens WhatsApp and pre-fills chat with text `BOOK_AT_{businessId}`
- Deep link works correctly on both iOS and Android devices
- Link uses business phone number from business profile
- Fallback handling if WhatsApp is not installed
