## Dev Notes

### Backend Architecture Context [Source: architecture/12-backend-architecture.md]

**Authentication Flow Implementation:**
```typescript
// ClerkAuthGuard Structure
@Injectable()
export class ClerkAuthGuard implements CanActivate {
  async canActivate(context: ExecutionContext): Promise<boolean> {
    const request = context.switchToHttp().getRequest();
    const sessionToken = request.headers.authorization?.replace('Bearer ', '');
    
    if (!sessionToken) {
      throw new UnauthorizedException('No session token provided');
    }
    
    try {
      const claims = await Clerk.sessions.verifySession(sessionToken);
      request.auth = claims; // Attach user claims to request
      request.user = await this.syncUserToDatabase(claims.userId);
      return true;
    } catch (err) {
      throw new UnauthorizedException('Invalid session token');
    }
  }
}
```

### WhatsApp Integration Context [Source: architecture/07-external-apis.md, architecture/08-core-workflows.md]

**WhatsApp Authentication Requirements:**
- Webhook endpoint: `POST /webhooks/whatsapp` (public but signature-verified)
- Merchant verification through business phone number mapping
- Session management for merchant-customer interactions via Redis
- Business context attachment to WhatsApp message flow

**Integration Pattern:**
```typescript
// WhatsApp Webhook with Merchant Context
@Post('webhooks/whatsapp')
async handleWhatsAppWebhook(
  @Body() payload: WhatsAppWebhookPayload,
  @Headers('x-hub-signature-256') signature: string
) {
  // Verify webhook signature
  this.verifyWebhookSignature(payload, signature);
  
  // Extract business context from message
  const businessId = await this.extractBusinessFromMessage(payload);
  
  // Process with merchant context
  return this.whatsappService.processMessage(payload, businessId);
}
```

### Data Models Integration [Source: architecture/04-data-models.md]

**User Model Database Schema:**
```prisma
model User {
  id           String   @id @default(cuid())
  clerkUserId  String   @unique @map("clerk_user_id")
  phoneNumber  String   @map("phone_number")
  createdAt    DateTime @default(now()) @map("created_at")
  updatedAt    DateTime @updatedAt @map("updated_at")
  businesses Business[]
  @@map("users")
}
```

**User Sync Logic:**
