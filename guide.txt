  --------------

   Kill Server Commands

  Quick Kill (What I just did)

  # Find process using port 3000
  lsof -ti :3000

  # Kill the process (replace 99577 with the actual PID)
  kill -9 99577

  One-liner to kill server on port 3000

  kill -9 $(lsof -ti :3000)

  More Gentle Approaches

  # Try graceful shutdown first
  kill $(lsof -ti :3000)

  # If that doesn't work, force kill
  kill -9 $(lsof -ti :3000)

  Kill All Node Processes (Nuclear Option)

  pkill -f node
  # or
  killall node

  Check What's Running on Port 3000

  # See detailed info about what's using port 3000
  lsof -i :3000

  # Check all ports in use
  netstat -tulpn | grep LISTEN
